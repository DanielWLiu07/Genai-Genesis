# Fonts

Place custom fonts here (e.g., manga-style fonts for text overlays).

Recommended free manga/comic fonts:
- **Bangers** (Google Fonts) - Bold comic-style
- **Permanent Marker** (Google Fonts) - Hand-drawn style
- **Anime Ace** - Classic manga speech bubble font
- **CC Wild Words** - Comic lettering style

These fonts will be used for:
1. Text overlays in trailer clips
2. Title cards
3. UI accents for manga/comic projects
